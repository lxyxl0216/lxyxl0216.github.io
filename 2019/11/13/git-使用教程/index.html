git 使用教程

发表于 2019-11-13 更新于 2020-04-02 分类于 教程
本文字数： 680 阅读时长 ≈ 1 分钟

配置user信息

  • Git config —global user.name ‘your_name’
  • git config —global user.email ‘email’

git config —local 支队某个仓库有用

git config —system 对系统所有登录用于使用

—list 显示配置

建git仓库

把已有的项目纳入git管理

  • cd 目录
  • git init

新建项目使用git管理

  • cd 目录
  • git init 项目 （当前路径下创建项目的目录）
  • cd 项目

操作

git status 显示状态

git checkout master （切换到master分支）

git branch -av（当前在哪个branch里）

添加

git add —all（当前目录所有文件）

git add file （添加file文件）

git commit -am ‘———’

重命名

git mv old-file new-file

git commit -am’——‘

日志

git log 显示日志

git log —online

git log -4（显示最近的4个日志）

git log —all —graph 所有分支的演进历史

git help —web log （查看帮助）

提交提交

git commit —amend（修改最近一次的message）

删除当前commit文件

git rm file

备份

git clone —bare

Github

配置

创建ssh key ：ssh-keygen -t rsa -b -C “email”

添加key ：id_rsa.pub

关联

git remote add 《name》 url

git remote -v 查看分支

git fetch master 把master远端的拉下来，与本地无关联

git push —all
